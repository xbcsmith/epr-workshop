# Install Python

## Overview

In this section we will install python and get started.

## MacOS

```bash
brew install python3
```

```bash
brew install virtualenv virtualenvwrapper pyenv
```

## Linux

### Fedora

```bash
sudo dnf install -y \
    python3-devel \
    python3-pip \
    python3-pyyaml \
    python3-ruamel-yaml \
    python3-ruamel-yaml-clib \
    python3-jinja2  \
    python3-tools  \
    python3-virtualenvwrapper \
    python3-virtualenv \
    python3-pytest \
    python3-tox \
    ruff
```
